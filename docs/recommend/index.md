## 🤖 软件 

### [MissionBackup](https://github.com/Hellager/MissionBackup)

本地备份软件：

- 基于 Tauri，兼顾了 web 前端和 rust 后端
- 5MB 的安装包，15MB 的程序，多平台可用
- 基于 cron 和 notify 实现对目标定时/监控备份
- 可自定义备份保存时长和空间
- 通过图表直观展示备份情况
- 完善的文档支持
